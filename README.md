# Smart Lender AI

## Project Structure

The project is organized into the following phases:

1. Ideation and Brainstorming
2. Requirement Analysis
3. Project Design Phase
4. Project Planning Phase
5. Project Development Phase
6. Project Testing
7. Project Documentation
8. Project Demonstration
9. SmartLender (Project Code)

## Description

Smart Lender AI is a machine learning-based loan eligibility prediction system. It analyzes applicant details and predicts whether a loan application is likely to be approved.
